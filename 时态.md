时：事情发生的时间 -> 过去将来，过去，现在，将来

态：事情发生的状态 ->  一般，进行，完成，完成进行



|      | 一般                                              | 进行                          | 完成                         |
| ---- | ------------------------------------------------- | ----------------------------- | ---------------------------- |
| 现在 | 一般现在<br />will do <br />is/am/are going to do | 现在进行<br />Will be doing   | 现在完成<br />Will have done |
| 将来 | 一般将来<br />Do                                  | 将来进行<br />am/is/are doing | 将来完成<br />Have/has done  |
| 过去 | 一般过去<br />Did                                 | 过去进行<br />was/were doing  | 过去完成<br />Had done       |



- **一般现在**

  - 我是个学生 *I am a student.*
  - 他/她是个学生 *He/She is a student.*
  - 他们是学生 *They are students.*

- **一般将来**

  - **be going to do sth**
  - 我一会去写作业  
    - *I am going to do homework.*
    - *He/She is going to do homework.*
    - *They are going to do homework.*

-  **一般过去**

  - 某个在当前时间之前发生的事情,并且此事和现在没啥关系
  - 我以前是个学生
    -  *I was a student.*
    - *He/She was a student.*
    - *They were stutdents.*

- **现在进行**

  - 此时正在做的事情(持续的)
  - **be doing sth**
  - 我正在写作业 
    - *I am doing homework.*
    -  *He/She is doing homework.*
    - *They are doing homework.*

- **将来进行**

  - 之后的现在我在做的事情
  - **sb will be doing sth**
  - 我明年的今天应该在准备考试 
    - *I will be preparing for the exam.*
    - *He/She will be preparing for the exam.*
    - *They will be preparing for the exam.*

- **过去进行**

  - 同时是过去和进行的时态
  - 昨天这时候我正在写作业  
    - *I was doing homework yesterday at this time.*
    - *He/She was doing homework yesterday at this time.*
    - *They were doing homework yesterday at this time.*

- **现在完成**

  - 表示一件事情在过去某个时间已经完成了,而且此事对现在有一定影响
  - 我已经写完作业了 
    - *I have finished my homework.*
    - *He/She has finished homework.*
    - *They have finished homework.*

- **将来完成**

  - 以后的某个时间某件事已经完成(20年后我应该有小孩了)
  - 明天下午我应该已经写完作业了
    - *I will have finished homework by tomorrow afternoon.*
    - *He/She will have finished homework by ...*
    - *They have finished homework by ...*

- **过去完成**

  - 表示某件事发生在某件在发生的事情之前
  - 昨天我就已经写完作业了(写作业发生在昨天以前)
    - *I had finished homework.*
    - *He/She had finished homework.*
    - *They had finished homework yesterday.*
